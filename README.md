# 📖 Overview

The **EV Market Forecast Dashboard** is a Data Analytics and Machine Learning project developed to analyze historical Electric Vehicle (EV) market trends and forecast future EV sales.

The project combines **Machine Learning**, **Data Visualization**, and **Interactive Dashboards** to help users understand EV market growth, government support, charging infrastructure development, and global adoption trends.

The dashboard provides meaningful insights for researchers, policymakers, investors, manufacturers, and EV enthusiasts through interactive visualizations.

---

# 🎯 Project Objectives

- Analyze historical EV sales trends.
- Forecast future EV sales using Machine Learning.
- Study the impact of government subsidies on EV adoption.
- Visualize charging station infrastructure growth.
- Display country-wise EV adoption using an interactive world map.
- Generate actionable insights into future EV market opportunities.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/EV-Market-Forecast-Dashboard.git
```

Navigate to the project directory

```bash
cd EV-Market-Forecast-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the dashboard

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:8050/
```
